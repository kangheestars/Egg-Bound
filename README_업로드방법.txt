- 렉 감소: 전체 화면을 매번 다시 그리지 않고 현재 보고 있는 탭 위주로 갱신합니다.

업로드 방법:
1. 이 ZIP을 압축 해제합니다.
2. GitHub 저장소에 기존 파일을 덮어쓰기 업로드합니다.
3. index.html, manifest.json, service-worker.js, icon-192.png, icon-512.png가 저장소 root에 있어야 합니다.
4. Commit changes를 누릅니다.
5. GitHub Pages 주소에서 새로고침합니다.

주의:
- 웹이 예전 버전으로 보이면 브라우저 새로고침을 한 번 더 하거나, 홈화면 앱을 완전히 닫았다가 다시 켜세요.
