Egg Bound 웹게임 업로드용 패키지 v119 Game Guide

이번 업데이트:
- 상단 빠른 메뉴의 “저장” 버튼을 제거하고 “게임 설명서” 버튼을 추가했습니다.
- 게임 설명서에는 시작 방법, 골드/오프라인 수익, 알/부화, 진화, 광산, 유물 시세, 명성/칭호, 저장/불러오기를 정리했습니다.
- 수동 저장/백업 기능은 설정 탭에 그대로 유지됩니다.
- 업데이트 뉴스는 최신 소식인 게임 설명서 추가만 표시됩니다.
- service-worker.js 캐시 이름을 egg-bound-v119-guide-upload-ready-1 로 갱신했습니다.

업로드 방법:
1. 이 ZIP을 압축 해제합니다.
2. GitHub 저장소에 기존 파일을 덮어쓰기 업로드합니다.
3. index.html, manifest.json, service-worker.js, icon-192.png, icon-512.png, apple-touch-icon.png가 root에 있어야 합니다.
4. Commit changes를 누릅니다.
5. GitHub Pages 주소에서 새로고침합니다.

주의:
- 홈화면 앱은 캐시 때문에 몇 분 늦게 반영될 수 있습니다.
- 게임 저장은 같은 주소/같은 브라우저 기준으로 유지됩니다.
